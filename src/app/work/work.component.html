E-Pos System

I developed E-Pos System frontend mobile apps using Ionic and AngularJS. I mainly develop UI based on given design and implement code logic, price and tax calculation into the mobile apps. Besides, I integrated with backend-API into my mobile apps to fetch data from server side, and integrated with hardware printer to print report and receipt slips.

Labyrinth
